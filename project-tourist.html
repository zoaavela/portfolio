Site Touristique

Un site qui présente ma ville natale, avec une approche centrée sur l'accessibilité et l'expérience utilisateur. (projet de premiére année)

Code Source Voir le site

Le Projet

Objectifs

Créer une plateforme touristique immersive pour promouvoir le patrimoine local. L'objectif était de dépasser le simple site d'information pour proposer une véritable expérience de découverte avant-voyage.

Stack Technique

  • • HTML5 (Sémantique)
  • • CSS3 (Flexbox/Grid)
  • • JavaScript (Vanilla)

Retours d'expérience

Ce que j'ai appris

J'ai approfondi ma maîtrise de la gestion d'états dans React et appris à optimiser le rendu de cartes interactives lourdes. L'accessibilité (a11y) était aussi au cœur de la conception.

Améliorations possibles

Avec le recul, j'aurais opté pour Next.js pour un meilleur référencement (SEO) et utilisé TypeScript pour sécuriser les données de l'API.
