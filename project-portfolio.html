Réservation Universitaire

Objectifs

Digitaliser l'ancien système papier de réservation de salles. Le but était de réduire les conflits d'horaires et de simplifier la vie administrative.

Stack

PHP (MVC), MySQL, JavaScript, Bootstrap.

Apprentissage

Ce projet m'a appris la rigueur de la conception de base de données (MCD/MLD) et l'importance de sécuriser les formulaires contre les injections SQL.

Améliorations

Une API RESTful aurait été plus adaptée pour permettre une future application mobile, au lieu du rendu côté serveur.
